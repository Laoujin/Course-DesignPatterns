Setup for Talk
==============

## Install
```
git clone https://github.com/itenium-be/Talk-DesignPatterns
cd Talk-DesignPatterns
npm install
git checkout v1-hard-coded
cd JavaScript
```


## Run

```
# Start Angular
cd impl-angular
npm start

# Start React
cd impl-react
npm start
```

## Time travel

```
# Initial codebase: Entire form hardcoded
git checkout v1-hard-coded

git checkout v2-from-config
```
